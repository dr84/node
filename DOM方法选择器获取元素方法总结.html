DOM 基础知识

DOM是 document object model(文档对象模型)的缩写,通过DOM可以操作文档内容

最常用的三种 node 节点类型:Document 文档节点,Element 元素节点,Text 文本节点

获取元素的方法

  • 通过id名选择器选择页面元素
    • document.getElementById('id');
  • 通过class名选择器选择页面元素
    • document.getElementByClassName('classname');
  • 通过tagName选择器选择页面元素
    • document.getElementsByTagName('tagname');
  • 通过name选择器选择页面元素
    • document.getElementById('name');

    通过css选择器选择页面元素

    id 选择器: class 选择器: 标签选择器: 在 js 中也可以用 css 选择器选择元素
    document.querySeletor('相应的css选择器名')
    document.querySeletorAll 选择('相应的css选择器名')

注意:

  • 1-驼峰写法的大小写
  • 2-通过标签名和类名获取的是一个由所有符合选择条件得元素的数组,而不是单独一个元素
